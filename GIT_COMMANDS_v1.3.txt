# Comandos Git para Subir English Memory v1.3.0

## Actualización a v1.3.0

# 1. Ver estado actual
git status

# 2. Agregar todos los cambios
git add .

# 3. Hacer commit con los cambios de v1.3.0
git commit -m "feat: Release v1.3.0 - TTS, Tema, Backup, Verbos

- Pronunciación TTS con pyttsx3 (botones en Vocabulario y Práctica)
- Toggle tema claro/oscuro con botón en header
- Backup automático cada 5 minutos (mantiene últimos 10)
- Nueva pestaña Verbos con 100 verbos irregulares
- Nueva pestaña Conjugación con 6 tiempos verbales
- Estadísticas mejoradas con contador de backups
- Sección de novedades en pestaña Ayuda
- Actualizada documentación completa (README, CHANGELOG)
- Versión actualizada a 1.3.0 en todos los archivos"

# 4. Subir cambios
git push origin main

# 5. Crear tag para v1.3.0
git tag -a v1.3.0 -m "Release version 1.3.0 - TTS, Tema, Backup, Verbos"

# 6. Subir el tag
git push origin v1.3.0

# 7. Crear Release en GitHub
# - Ir a GitHub → Releases → New Release
# - Tag: v1.3.0
# - Title: English Memory v1.3.0 - TTS, Tema, Backup, Verbos
# - Descripción: Copiar contenido de RELEASE_NOTES_v1.3.md
# - Adjuntar: dist/English Memory.exe (después de compilar)
# - Publicar

## Comandos para crear ejecutable antes del release

# Windows
python build_exe.py

# Linux
chmod +x build_linux.sh
./build_linux.sh

## Comandos útiles

# Ver historial
git log --oneline

# Ver diferencias
git diff

# Ver tags
git tag -l

# Ver último commit
git show

# Ver archivos modificados
git diff --name-only
